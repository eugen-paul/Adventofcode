## Move G to node-x0-y0

Now we must move `G` to the node `node-x0-y0`.
To move `G` one node left we need to do `5` steps (see example).
In total, the point must be moved by `31` steps.
So with my input data it takes `50 + 31*5 = 205` steps to move `G` to `node-x0-y0`.
